Groups & Commands

For those of you who don't know, Bukkit has been going through legal things over the past year. This has essentially haulted most development on popular plugins (and makes bukkit much harder to obtain). So, There are only 2 plugins - PermissionsEx and a little suite I wrote up. If you think I should add an ability, let me know, I can probably make it work.

While in game, you can see most of your available commands by using "/help SudoCraftSuite". Each rank inherits the previous ranks' abilities.

Guest

Guests cannot build or interact with the world. They need to be promoted to builder to begin play.

Commands

/tpr [player] Send a teleport request to another player.
/tpa [name] Accept a teleport request from another player.
/back Teleports you to the last place you teleported from.
/vote Vote to change the time and weather. 80% of players need to vote. Valid options are: sun, rain, storm, day, and night.
/spawn Teleports you to the spawn location

Builder

Builders are the main group and can do most things.

Commands

/sethome Set a home to teleport back to later. You can specify a name as an argument to have multiple homes.
/home Teleport to a home. Using no argument will assume your default home.
/homes This will list all homes you have saved.
/delhome Deletes the specified home. No argument assumes your default home.
/builder [player] Promote a Guest to Builder. Please, only promote people you know.
/warp [name] Teleports you to a global warp location.
/warps Lists all available warps.

Admin

Admins are the head honchos.

Commands

/kick Kicks a player from the server
/ban, /unban Permanently ban or unban a player from the server
/setspawn Sets the spawn location for the world your in to where you're standing.
/setwarp Allows you to create a public warp locations.
/delwarp Deletes a public warp locations.
